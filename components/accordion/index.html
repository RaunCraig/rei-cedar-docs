Accordion

Why is this text here?

Vertically-stacked list that allows users to expand and collapse additional content. I assume this will be edited to be a proper sentence.

  • Overview
  • Design Guidelines
  • API
  • History

Accordions are built from two components, cdr-accordion and cdr-accordion-item.

# Props

# cdr-accordion

compact

name

boolean

type

false

Default

Set compact style of cdr-accordion-item child components.

borderAligned

name

boolean

type

false

Default

Set border-aligned style of cdr-accordion-item child components.

showAll

name

boolean

type

false

Default

Set all child cdr-accordion-item components to display open by default.

# cdr-accordion-item

id

name

string

type

n/a

Default

Required id for component reference. Id must be unique.

label

name

string

type

n/a

Default

Set the readable text on the cdr-accordion-item button or trigger. Required.

show

name

boolean

type

false

Default

Set a single cdr-accordion-item to display open by default. Prop showAll will take precedence, if true.

# Slots

# cdr-accordion

default

name

innerHTML on the inside of the accordion component

# cdr-accordion-item

default

name

Default slot for cdr-accordion-item content.
